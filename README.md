# Room-Wise-UPS-Integrated-Residential-Power-System
Design and simulation of a Room-Wise UPS-Integrated Residential Power System for a complete house with 11 rooms and 13 appliances. Electrical parameters calculated in MATLAB, circuit simulated in NI Multisim 14.2. Total load: 8945W | UPS load: 1395W. CEP project — PIEAS, Islamabad.
